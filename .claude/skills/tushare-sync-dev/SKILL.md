# Tushare 文档查询

## 核心原则

**接口名称、参数名称、字段名称、金额单位必须以官方文档为准，禁止凭猜测、变量名、历史代码或邻近接口类推。**

## 使用方式

通过 skill 目录下的 Python 脚本查询文档（不使用 WebFetch）：

```bash
# 查询某个接口的完整文档
python <skill-dir>/fetch_tushare_doc.py <接口名>

# 按关键词搜索接口
python <skill-dir>/fetch_tushare_doc.py --search <关键词>

# 列出所有接口
python <skill-dir>/fetch_tushare_doc.py --list
```

## 查询流程

### 第一步：定位接口

- 用户明确给出接口名 → 直接查询
- 不确定接口名 → 用 `--search` 按关键词搜索，或读取 `<skill-dir>/references/数据接口.md` 浏览分类

### 第二步：阅读文档（必须完成）

运行脚本获取文档后，确认以下信息：

| 要确认的项 | 说明 |
|----------|------|
| 接口名 | 与文档标题/示例完全一致 |
| 入参 | required / optional，字段名一字不差 |
| 输出字段 | 实际返回的列名、数据类型 |
| **金额单位** | 万元 / 亿元 / 元，不同接口可能不同 |
| 所需积分 | 确认当前 7000 积分是否满足 |
| 单次返回上限 | 是否需要分页/循环 |
| 更新时间 | 每日几点后可取当日数据 |

**如果文档无法访问，必须告知用户并暂停，不得继续编码。**

### 第三步：输出摘要

将文档中的关键信息整理为结构化摘要返回给用户，包括入参表、输出字段表、注意事项。
