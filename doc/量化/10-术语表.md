# 10 术语表

↩ [返回索引](00-index.md)

---

| 术语 | 中文 | 含义 |
|---|---|---|
| **PIT** | Point-in-Time | 数据按时间点严格还原，不偷看未来 |
| **IC / RankIC** | 信息系数 / 秩相关 IC | 因子值与未来收益的（秩）相关性 |
| **IR** | 信息比率 | IC 均值 / IC 标准差 |
| **NDCG** | 归一化折损累积增益 | 排序质量指标，截断在 Top-N |
| **Embargo** | 禁运期 | CV 中 train 与 val 之间的隔离窗口 |
| **PSI** | Population Stability Index | 衡量两期分布漂移 |
| **MDD** | Max Drawdown | 最大回撤 |
| **LambdaRank** | — | 直接优化 NDCG 的排序学习方法 |
| **Triple Barrier** | 三重屏障 | López de Prado 的标签方法 |
| **Forward Return** | 远期收益 | 训练标签的基础 |
| **Universe** | 股票池 | 当期可交易的股票集合 |
| **Survivorship Bias** | 幸存者偏差 | 只用现存股票回测引入的偏差 |
| **Cross-Sectional** | 横截面 | 同一时点不同股票的比较 |
| **Walk-Forward** | 滚动前进 | 时序数据的 CV 方法 |
| **GBDT** | 梯度提升决策树 | 串行 boosting 的树模型族 |
| **GAM** | 广义加性模型 | 各因子贡献可单独画出的可解释线性族 |
| **VWAP** | 成交量加权平均价 | 实盘成交基准价 |
| **OOS / IS** | 样本外 / 样本内 | Out-of-Sample / In-Sample |
| **SHAP** | Shapley 加性解释 | 树模型每个特征对单次预测的边际贡献 |
| **Strategy-aware labeling** | 策略感知标签 | 按真实交易出场规则模拟生成标签的方法 |
