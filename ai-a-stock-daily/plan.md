# A股每日AI复盘自动化系统 - 执行计划

## 目标
构建一套完整的A股每日AI复盘自动化流程，产出微信公众号文章+微信视频+抖音视频三件套。

## 参考文章结构反推
1. 标题：YYYY-MM-DD 复盘以及MM-DD日前瞻：核心结论概括
2. 开篇：AI生成声明 + 免责说明 + 特别说明
3. 一、先给结论（核心线+下一交易日判断+资金切换路径）
4. 二、大盘全景数据（指数/成交/涨跌分布/情绪）
5. 三、重点板块拆解（表现+个股+逻辑+持续性+前瞻）
6. 四、潜力板块跟踪
7. 五、盘后/隔夜信息：美股映射
8. 六、宏观与政策消息面
9. 七、综合结论与策略建议
10. 八、重点个股观察池（分仓管理）
11. 九、最实战的结论

## 执行阶段

### Stage 1: 数据采集（并行）
- **任务1A**: A股核心数据抓取（Tushare）
  - 大盘指数（上证/深证/创业板/科创50）
  - 涨跌停统计、涨跌分布
  - 板块涨跌幅排行（概念+行业）
  - 个股资金流向（主力净流入排行）
  - 强势股/龙虎榜数据
  - 当日成交额TOP个股
- **任务1B**: 美股盘前数据抓取
  - 美股三大指数期货/盘前
  - 热门中概股表现
  - 芯片/AI相关美股表现
- **任务1C**: 消息面采集
  - 财经新闻热点（模拟）
  - 政策面消息

### Stage 2: 内容生成
- 基于Stage 1数据，生成5000-10000字复盘文章
- 彭博终端/高盛研报风格
- 深靛蓝 #1A237E + 警示橙 #FF6D00 配色
- 完整九段式结构

### Stage 3: 视频生成（并行）
- **任务3A**: 微信视频（横屏 16:9，3-5分钟）
  - 文字配音（男音，专业可信风格）
  - 麦肯锡配色
  - "仅用于学习研究，不构成投资建议"免责
- **任务3B**: 抖音视频（竖屏 9:16，3-5分钟）
  - 同款配音
  - 麦肯锡配色
  - 同款免责

### Stage 4: 部署交付
- 构建Web界面（React）
- 一键手动触发按钮
- 定时自动运行配置
- 文章预览 + 视频下载
- 配色方案：#1A237E + #FF6D00

## 技术栈
- 数据采集: Tushare (A股) + yfinance (美股)
- 内容生成: LLM
- 视频: 生成语音+合成视频
- 部署: React + Vite 静态站点
- 交付: 静态HTML（可部署到任意平台）
