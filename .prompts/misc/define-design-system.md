# Ember Studio

## 概述
一套温暖、注重工艺感的设计系统，专为创意项目管理工具打造。视觉风格将赤陶土的温暖与现代极简主义融为一体——柔和的大地色调奠定界面基调，琥珀色高亮点缀操作与进度。专为兼顾审美与效率的团队而设计。浅色和深色模式都经过精心设计，而非简单反色。整体氛围平静、专注，带有一丝低调的质感。

## 颜色
- **主色** (#C2410C)：赤陶土色 — 用于行动按钮、激活状态、链接、焦点环、进度指示器
- **主色悬浮** (#9A3412)：烧赭色 — 主元素的悬浮状态
- **强调色** (#F59E0B)：琥珀色 — 通知、徽章、高亮、新条目指示器
- **中性色** (#78716C)：石色 — 次要文字、占位符、时间戳、元数据
- **背景色** (#FAFAF9)：暖白色页面背景，带有一丝奶油感
- **表面色** (#F5F5F4)：卡片、面板、模态框 — 略带暖意的米白色
- **浮起表面** (#E7E5E4)：悬浮状态、选中行、激活标签页
- **主文字色** (#1C1917)：暖近黑色 — 标题、正文、主要标签
- **次要文字色** (#57534E)：暖灰色 — 描述、说明、次要信息
- **边框色** (#D6D3D1)：暖灰色边框 — 卡片边缘、分割线、输入框边框
- **成功色** (#16A34A)：已完成任务、已批准项目、正向状态
- **警告色** (#D97706)：即将到期、需要关注、警示横幅
- **错误色** (#DC2626)：已逾期、失败、危险操作

## 字体排版
- **展示字体**：Playfair Display — 从 Google Fonts 加载
- **正文字体**：Source Sans 3 — 从 Google Fonts 加载
- **代码字体**：Fira Code — 从 Google Fonts 加载

展示文字和标题使用 Playfair Display 粗体，字间距 -0.02em。衬线展示字体传递工艺感与用心态度。正文和界面文字使用 Source Sans 3，字重为常规（400）和半粗（600）——简洁、高可读性的无衬线字体。代码块使用 Fira Code，启用连字功能。

字号体系：展示 64px、大标题 48px、章节标题 28px、副标题 20px、正文 16px、小字 14px、说明文字 12px、标注行 11px 大写宽字距。

## 层次阴影
卡片平铺时带有 1px 暖色边框（#D6D3D1），悬浮时获得柔和阴影（0 4px 16px rgba(28,25,23,0.06)）。激活/选中的卡片左侧显示 2px 赤陶土色边框。主按钮悬浮时获得暖色光晕（0 4px 12px rgba(194,65,12,0.25)）。模态框使用更大的阴影（0 24px 48px rgba(28,25,23,0.12)）配合背景模糊。导航栏透明并带背景模糊，滚动时出现 1px 底部边框。

## 组件规范
- **按钮**：主按钮使用赤陶土色（#C2410C）填充、白色文字、8px 圆角、半粗字重。次要按钮使用透明背景、1px 石色边框（#D6D3D1）、暖色文字。幽灵按钮无边框，仅有文字颜色。危险按钮使用红色背景、白色文字。所有按钮过渡时间 150ms。尺寸：小（高 32px，内边距 12px）、中（40px，16px）、大（48px，24px）。
- **卡片**：暖白色表面（#F5F5F4），1px 边框（#D6D3D1），12px 圆角，16px 内边距。项目卡片左侧显示 4px 彩色竖条，颜色与项目指定色一致。悬浮时上移 2px 并加深阴影。选中卡片显示赤陶土色左边框。
- **输入框**：1px 边框（#D6D3D1），表面色背景（#F5F5F4），8px 圆角，12px 内边距，16px 字号。聚焦时边框变为赤陶土色并显示暖色光环（0 0 0 3px rgba(194,65,12,0.12)）。错误时边框变红。标签在输入框上方，14px 半粗字重。
- **标签片（Chips）**：胶囊形状（9999px 圆角）。分类标签片：石色-100 背景、石色-600 文字、6px/14px 内边距。激活状态：赤陶土色背景、白色文字。优先级标签片使用语义色配对应文字色。
- **进度条**：高度 4px，完全圆角，石色-200 轨道，赤陶土色填充。填充动画 300ms ease 过渡。上方显示小字百分比标签。
- **头像**：圆形（9999px 圆角），默认尺寸 32px。团队展示时以 -8px 间距叠放。边框：2px 纯色表面色，用于分隔。
- **标签页**：水平排列，下划线风格。未激活：石色文字，无下划线。激活：赤陶土色文字配 2px 底部边框。悬浮：暖灰色背景。
- **导航栏**：侧边栏布局，宽度 256px。暖白色背景配 1px 右边框。顶部显示 Logo 和工作区名称。带箭头切换的可折叠分组。激活项：3px 赤陶土色左侧强调条配暖色调背景。

## 间距
- 基础单位：4px
- 间距体系：4、8、12、16、20、24、32、40、48、64、80px
- 组件内边距：小 8x12、中 12x16、大 16x24
- 区块间距：移动端 24px、平板 32px、桌面端 48px
- 容器最大宽度：1200px，水平内边距 24px
- 卡片网格间距：移动端 16px、桌面端 24px

## 圆角
- 4px：行内代码、小型徽章
- 8px：按钮、输入框、下拉选择框、下拉菜单
- 12px：卡片、面板、模态框、弹出框
- 9999px：头像、标签片、胶囊元素、进度条

## 规范与禁忌
- 应当：赤陶土色（#C2410C）仅用于交互元素和激活状态，切勿作为装饰
- 应当：严格遵循 4px 间距网格
- 应当：标题使用 Playfair Display，正文使用 Source Sans 3——衬线与无衬线的对比是本设计系统的标志性风格
- 应当：保持暖色调一致性，避免使用冷灰色或偏蓝的中性色
- 应当：琥珀强调色仅用于需要吸引注意的元素，保持克制
- 禁止：单个页面使用超过两种字重（常规 + 半粗）
- 禁止：混用圆角值——按钮用 8px，卡片用 12px
- 禁止：使用纯黑或纯白——始终使用暖色调色板中的值
- 禁止：添加装饰性元素——温暖感来自色彩，而非装饰
- 禁止：在同一区块放置多个赤陶土色按钮——每个视图只有一个主要行动按钮
